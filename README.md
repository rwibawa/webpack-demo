# webpack-demo

an exercise to learn _webpack_.

[Getting Started](https://webpack.js.org/guides/getting-started/)

## Basic Setup

```sh
$ mkdir webpack-demo
$ cd webpack-demo

$ node -v
v14.15.3

$ npm init -y
$ npm install webpack webpack-cli --save-dev
```
